Package Specification

V-Octopus Web Server is intended to specify the W3's specification on Http/1.1.
  • Usage of Java 5 Executors for Multi-threading capabilities
  • Usage of AspectJ 5 for the implementation of cross-cutting concerns such as Loggin, Degub and internal caching mechanisms.
