Handling the incoming request depends on the type of the resource: this package contains the handlers necessary to generate th output for the following resource's requests:
  • Ascii files;
  • Binary files;
  • CGI execution (Python, Ruby, Perl, ...), anything the OS can execute through command line;
  • Directory Fancy listing;
  • Cached, Protected files;
  • The following error messages: 401, 413, 404, 500 ;
  • Web Service request (Still under development)
