When it comes to the validation of the incoming requests, the validation package contains an implementation of the Interpreter Design-Pattern, that holds breaks the tokens of the incoming requests into a series of tokens, and then validates according to the specification. This information is passed to the HttpRequest instance and also to the Request Handler.
